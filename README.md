# Computational_moddeling_HOS2
# Breakwater Caisson FEM

2D plane-strain finite element model of a sand-filled breakwater caisson under irregular wave loading.  
Built for CIEM4220 Computational Modelling — TU Delft, Group HOS2.

## Usage

Open `FEM_caison_def.ipynb` and run all cells

## Dependencies

`numpy` · `scipy` · `matplotlib` · `meshio` · `gmsh` · `pandas`
